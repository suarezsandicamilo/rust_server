# Rust_Server

A web server with a simple To Do app implemented with Rust.

## Install

Clone the repository

```
git clone https://github.com/suarezsandicamilo/rust_server.git
```

## Usage

Run with [Cargo](https://crates.io/crates/deno)

```
cargo run
```
